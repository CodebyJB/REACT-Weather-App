# ☀️ Weather App ☂️

This is a simple weather application built using React that allows users to search for the weather forecast of a specific location. The app retrieves geolocation data using a search query and then fetches weather forecast data for that location.

## Features

- Search for a location to get weather forecasts.
- Display weather forecast data for the upcoming days.
- Weather icons are displayed based on the weather code from the API response.
- Shows the maximum and minimum temperatures for each day.
- Provides a user-friendly interface.

## Installation

1. Clone this repository to your local machine using:
```

git clone https://github.com/your-username/weather-app.git

```

2. Navigate to the project directory:
```

cd weather-app

```

3. Install the dependencies using your preferred package manager (npm or yarn):
```

npm install

```

## Usage

1. Run the development server:
```

npm start

```
This will start the app on [http://localhost:3000](http://localhost:3000) in your browser.

2. Enter a location in the search input to get the weather forecast for that location.

## API Usage

The app uses the following APIs to retrieve weather and geolocation data:

- [Geocoding API](https://geocoding-api.open-meteo.com/v1) - Used to fetch geolocation data based on the search query.
- [Open-Meteo API](https://api.open-meteo.com/v1/forecast) - Used to retrieve weather forecast data based on the latitude, longitude, and timezone of the selected location.
